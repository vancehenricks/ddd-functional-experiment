# ddd-functional-experiment

Unfinished blog site with DDD in mind while trying to incorporate it with a functional.

## Setup

```bash
npm install
npm db:start
npm db:seed
```

## Lint

```bash
npm run lint
```

## Test

```bash
npm run test
```

## Development

```bash
npm run dev
```

## Linting sql files

```bash
pip install sqlfluff
```
